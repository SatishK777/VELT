# VΞLT Platform Upgrades - The Drop Page (3D Immersive Showcase)

## 1. Overview
The `/drop` route will feature the "3D Immersive Showcase." This bypasses traditional e-commerce scrolling grids in favor of an experiential cyber-exhibition. The page will exist in a locked full-viewport layout where horizontal scrolling operates a smooth carousel that sweeps through marquee assets, backed by massive brutalist typography.

## 2. Architecture & Layout
- **Container:** Pure full-viewport `h-screen w-full` with hidden vertical overflow.
- **Horizontal Interaction:** Vertical scrolling (the mouse wheel) will be mathematically mapped to lateral horizontal translation using Framer Motion's `useScroll` and `useTransform`.
- **Composition:** 
  1. A massive background layer of outlined tracking typography repeating incredibly fast (`marquee` style).
  2. The interactive slider layer, featuring giant high-rez cutouts of the drop items.
  3. A static, strict HUD overlay providing Navigation and Terminal metrics (Drop time, availability, structural IDs) in standard Cyber-Acid Volt Yellow and White.

## 3. Visuals & Typography
- **Core Aesthetic:** Pure Brutalist. 
- **Type Integration:** The name of the currently focused item will appear behind the item itself in outline or negative space typography, sliding perfectly with the model. 
- **Effects:** When switching items, a sharp GSAP tracking blur or glitch jitter will make it feel like an analog signal switching channels.

## 4. Implementation Direction
- Create `app/drop/page.tsx`.
- Refactor the codebase to utilize a continuous horizontal scroll layout for this page specifically.
- Import standard `components/Navbar.tsx` but render it with high z-index over the moving viewport.
- Introduce hover buttons that trigger the `[Secure Asset ->]` checkout flow.
